## Description

This repository is for test [Nest](https://docs.nestjs.com) with [Prisma](https://www.prisma.io)

[Nest](https://github.com/nestjs/nest) framework used.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
